# AI Revolution

A responsive landing page built with React and Tailwind CSS, following along with a YouTube tutorial.

🔗 **Live Demo**:(https://ai-revolution-landingpage.netlify.app/)

## Tech Stack

- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Ruaa-Maher-Faroun/ai-revolution.git
cd ai-revolution

# Install dependencies
npm install

# Start the development server
npm run dev
```

- Structuring a React project with reusable components
- Using Tailwind CSS utility classes for responsive layouts
- Building and deploying a static landing page
